# ChargeIt

## Overview
ChargeIt is an iOS application designed to help electric vehicle users discover nearby charging stations quickly and easily. The app focuses on a smooth onboarding experience and an intuitive home screen that highlights charging locations on a map.

## Technology Stack
- **Language:** Swift  
- **UI Framework:** UIKit  
- **UI Design:** Storyboards  
- **IDE:** Xcode 26.2  

## Current Status
The application currently implements the complete user interface using Storyboards. Core backend functionality and integrations are planned for future development.

## Implemented UI Features
- User registration using email, name, and phone number  
- Sign-in options including email/password, Google, and Facebook  
- Password recovery using a **Forgot Password** flow  

### Onboarding
- Optional electric vehicle type setup  
- Optional payment details for faster future usage  

### Home Screen
- Map view displaying nearby EV charging stations  
- Search functionality for specific stations  
- Filters based on charging power and connector type
